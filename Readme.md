### 🖥️ Project Showcase
> **Interactive Dashboard** designed for managing browser extensions with a focus on speed and clean UI.

| Feature | Description | Status |
| :--- | :--- | :---: |
| **🌓 Dark Mode** | Persistent theme switching using `localStorage` | ✅ Done |
| **🔍 Live Filter** | Real-time filtering (Active/Inactive) via TypeScript | ✅ Done |
| **📦 Data Fetch** | Dynamic content loading from `JSON` | ✅ Done |
| **📱 Responsive** | Fully optimized for Mobile & Desktop | ✅ Done |

---

### ✨ Highlights

* **Logic-First:** Built with **TypeScript Enums** and **Interfaces** for robust code.
* **Performance:** Zero-latency DOM manipulation using event delegation.
* **Styling:** Clean architectural CSS using **SASS** variables and nesting.
